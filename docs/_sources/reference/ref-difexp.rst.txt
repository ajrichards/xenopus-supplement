.. notes from the literature


Differential expression analysis
======================================

These results are specific to the :doc:`reference assembly <star-assembly>`.


Create raw and normalized count matrices
------------------------------------------

The software HTSeq was used to estimate read abundances [Anders14]_.

    * :download:`getCountMatrixReference.py <../assembly/getCountMatrixReference.py>`

Run edgeR and DESeq
----------------------

DESeq2 was run on the assembly for the gene count matrices [Love14]_.

    * :download:`runDeAnalysis.py <../features/runDeAnalysis.py>`
    * :download:`runDESeq.R <../features/runDESeq.R>`

Then the resulting \*.csv files were summarized using

    * :download:`summarizeResultsDE.py <../features/summarizeResultsDE.py>`
